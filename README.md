🚀 Amazon Prime Video – Power BI Dashboard Project
I recently built an interactive Power BI dashboard to analyze the content library and performance trends of Amazon Prime Video.
The dashboard provides insights into multiple aspects of the platform, including content distribution, user engagement trends, ratings, category performance, and global availability of titles.
The dataset used for this project was sourced from Kaggle, and the entire data pipeline involved several key steps such as data cleaning, transformation, data modeling, and creating DAX measures to generate meaningful insights.
Key highlights of the dashboard include:
📊 Analysis of movies and TV shows based on genre, release year, and ratings
🌍 Visualization of global distribution of content across different countries
⭐ Insights into content ratings and popularity trends
📈 Category-wise performance to understand which genres dominate the platform
Through this project, I strengthened my skills in Power BI, data modeling, DAX, and data visualization, while also gaining experience in transforming raw datasets into meaningful and actionable insights.
